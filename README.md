# WorkFlex App

## Descripción del proyecto

Aplicación web orientada a **flexibilidad laboral** o **gestión de disponibilidad** (turnos, preferencias de trabajo híbrido/remoto, etc.) con Next.js y Firebase.

## Problema que resuelve

Equipos y personas necesitan alinear expectativas de presencia y horarios sin infinitos threads: ofrece una capa dedicada para registrar y consultar reglas de flexibilidad en un solo lugar.

## Stack

- Next.js, React, TypeScript
- Firebase

## Requisitos

- Node.js LTS

## Instalación

```bash
npm install
npm run dev
```

Scripts: `build`, `start`, `lint`, `export`.

## Variables de entorno

`.env.local` con credenciales `NEXT_PUBLIC_FIREBASE_*` de Firebase.
